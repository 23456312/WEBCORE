¿Cuáles son las partes de un URL?

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

❥Get nos ayuda a solicitar alguna representación de algun recurso especifico. ❥ Head es similar a get en cuestio de que pide una respuesta sin embargo no utiliza el cuerpo de la respuesta. ❥Post Envia una entidad a un recusro especifico. ❥ Put nos ayuda a poder replazar las representaciones actuales con las deseadas. ❥ Patch nos ayuda a aplicar modificaciones parciales a un recurso. ❥ Delte borra algun recurso especifico. (Métodos De Petición HTTP - HTTP | MDN, 2023)

Por la cuestión de las diferencias esta la ejecución la cual Java se ejecuta en JVM lo cual lo hace independiente del sistema opertaivo sin embargo Java Script se ejecuta en los navegadores. Otra diferencia es que Java es un lenguaje con un tipado fuerte y estatico. Java script ofrece un tipado más dinamico y flexible.
